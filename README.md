# Calculator CLI App

This is a simple command-line calculator built using Python.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Exit option

## Concepts Used

- Functions
- Loops
- Conditionals
- User Input
- CLI Interaction

## How to Run

1. Install Python.
2. Open terminal in the project folder.
3. Run:

```bash
python calculator.py
```

## Example

Choose an operation, enter two numbers, and the calculator displays the result.
